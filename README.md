# 🚀 Core PHP Complete Learning Notes

A complete **Core PHP Learning Repository** containing beginner to advanced PHP concepts with simple explanations, examples, OOP concepts, MySQL CRUD operations, file handling, and REST API basics.

This repository is designed for students, beginners, and developers who want to learn PHP from scratch before starting the Laravel Framework.

# ▶️ How to Run

1. Clone the repository.
2. Copy the project into the `htdocs` folder (XAMPP).
3. Start Apache and MySQL.
4. Create the required MySQL database.
5. Import the SQL file (if available).
6. Open the project in your browser.

Example:

```
http://localhost/project-name/
```
